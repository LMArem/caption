## 关于 *archive.ps1* 脚本

脚本基于 PowerShell 语言, 有两种运行方式:

1. 使用 PowerShell 程序作为打开方式, 适合单独目录
2. 通过控制台调用, 适合批量目录

脚本所有运行参数 (运行方式不同, 部分参数不可用):

- -a 打包当前路径下所有目录
- -h 打印帮助信息
- -e 退出脚本
